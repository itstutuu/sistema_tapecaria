Projeto desenvolvido para a matéria de Projeto Integrador I do curso de Ciência de Dados.

Ele consiste de 3 partes principais:
Um banco de dados relacional, uma interface para cadastro de clientes e serviços, e um dashboard financeiro.

As principais ferramentas utilizadas foram PostgreSQL, linguagem Python com as bibliotecas PG8000 para conexão com o banco e Faker para testes, e o MS Power BI.

O projeto foi finalizado letivamente, mas a ideia é continuar o atualizando até que chegue em nível de produção.

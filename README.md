Projeto desenvolvido para a matéria de Projeto Integrador I do curso de Ciência de Dados.

Aqui jaz apenas a parte técnica do projeto, pois a parte de pesquisa e negócios é pouco significante para este espaço.
Ele consiste de 3 partes principais:
um banco de dados relacional, uma interface para cadastro de clientes e serviços, e um dashboard financeiro.

As principais ferramentas utilizadas foram PostgreSQL, linguagem Python com a biblioteca Faker para testes e MS Power BI.

O projeto foi finalizado letivamente, mas a ideia é continuar o atualizando, pois se encontra 75% pronto,
tendo que suas partes serem melhor integradas e refinadas para real usabilidade.
